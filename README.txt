
A continuación te explico como debe ejecutar el programa de recomendaciones.

1.- Modificar el archivo "vars.py" con tus respectivos codigos.

2.- Una vez haya instaladas las bibliotecas debe ejecutar el archivo "script.py".

3.- Una vez corriendo el programa le pedirá que inserte una URL de alguna cancion de spotify, les dejaré una como ejemplo si no tiene acceso a spotify por sus medios.
("https://open.spotify.com/intl-es/track/1JSTJqkT5qHq8MDJnJbRE1?si=23bd0fd789c2447f") Esta canción se llama "Every Breath You Take" de "The Police".

4.- Luego de ingresar la URL debe presionar enter.

5.- A continuación comenzará a examinar la canción con la ayuda de una API que provee Spotify Developer, obteniendo diversas caracteristicas de la canción las cuales le apareceran en el execute.

6.- Una vez deje de cargar dirijase al archivo "recommended_song.csv" dentro del proyecto.

7.- En ese archivo le aparecerán todas las canciones que el sistema le recomienda en base a la canción proporcionada.

8.- Adicionalmente esto crea una playlist con las canciones recomendadas en tu aplicación de Sporify.

A modo de aclaración esta comunicación entre la aplicación y la API se produce gracias al cliente-id y cliente-id-secret.
